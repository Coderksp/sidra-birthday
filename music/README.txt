Optional background music
=========================

To enable the soft background music:

1. Put an MP3 file here named exactly:  song.mp3
   (e.g.  E:\sidra-birthday\music\song.mp3)

2. Reload the site and click the 🔇 button in the top-right corner
   to play / pause it.

If no song.mp3 is present, the site works perfectly — the music
button simply has nothing to play.

Tip: pick a soft, royalty-free instrumental so it can be shared/hosted
without any copyright issues.
